Process

To start the project, we met at the labs. We started by getting just the breed from the API. Once we got the images onto the screen, we worked on styling the images so that they were all the same size and centered. We had a link under the image to open the full image in another tab, but we decided to turn the image itself into the link. After we styled the images, we styled the entire page. We found background images and an image for the button so that it would fit our theme. While styling, we decided to add another input field for the sub breed and limit. Once everything worked, we stored the inputed values in local storage.

Requirements Met

Local Storage
The webiste saves the last entered Breed, Sub Breed, and limit to the local storage. When the website is reloaded, the last search terms are autofilled into the fields
Controls
There are 3 input fields for the search terms of Breed, Sub Breed, and limit of images to show. There is also the "Fetch" button that, when clicked, get's the images from the Dog API
