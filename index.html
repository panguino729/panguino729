Within These Walls

Made using Unreal Engine (4.27) by The Tri-Hards (Corrine Fair, Gel Howell, Katarina Tretter)

Within These Walls is a first-person, psychological horror game that invites players to question their reality. Playing as Jordan, the player will be guided through an abandoned Victorian manor where they must piece together the truth of the past that has haunted their family. The manor has many secrets for you to discover – the information you find about the past has the power to influence the present by altering how Jordan understands what happened to their grandparents and how they perceive what’s happening to them now. Are the strange happenings all in their head? Or is the old Cromwell Manor truly haunted? It’s up to you to decide.

Responsibilities: In-game artifacts, narrative backstory, design and layout of the house

Book Editor Plugin

Made using Python and C#

Book Editor Plugin for Calibre made for IGME.690 (Tools/Extending Game and Media Systems). Allows the user to drag and drop an epub onto the plugin icon, which converts from epub to RTF and opens the file in the external Book Editor Tool. The external Book Editor Tool is a basic text editing tool that allows the user to edit RTF files. The user can change font, font size, and make text bold, italic, or underlined. The external tool also allows the user to choose where to save the file to, but it will default to the original RTF location.

Gameplay and Prototyping

Made using Unity (2020.3.14f1)

Gameplay and Prototyping

Made using Unity (2020.3.14f)

Gameplay and Prototyping focuses on the design methods, strategies, and to some extent tools used in the creation of video games. Over the course of a semester we made a total of 6 games each spanning about 2 weeks, with the last 2 games taking about 3 weeks each. For each game, we were put into different teams of 4/5. After the 1st week, we were required to turn in a structural prototype. At the end of the 2nd week, we took our prototype and playtested it with everyone else in the class before submitting our playtest notes and final build.
